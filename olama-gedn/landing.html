How It Works

Up and running
in minutes.

01

Download and install

Run the Inno Setup installer. Temporal Portal drops into your VST3 folder automatically with all WebView assets bundled inside the binary.

~4 MB · Single file · No C++ runtime deps
02

Scan in your DAW

Ableton, FL Studio, or Reaper will find it on the next plugin scan. Open the editor and the full HTML UI renders natively — no setup required.

Ableton · FL Studio · Reaper · Any VST3 host
03

Select your DAW profile

Switch the voicing profile to match your host. The DSP chain reconfigures its EQ, compressor, and modulation parameters to complement your DAW's processing chain.

Verified by deterministic smoke test · All profiles PASS
04

Dial in your sound, export your preset

Use Time Shift, Pitch Bend, Key, Detune, Sauce, and Wet/Dry to sculpt the exact timbre. Export a JSON preset to recall or share the session anywhere.
